## 핵심 데모 시나리오

### 시나리오 1: 일반 상담 흐름
```
사용자: 직장 스트레스와 불안 호소
  → Triage: Low Risk (2/10)
  → Counseling Agent: CBT 기반 인지 왜곡 탐색
  → 세션 요약 + 임상 노트 자동 생성
```

### 시나리오 2: 위기 개입 흐름
```
사용자: 대화 중 자해 의도 표현
  → Triage: High Risk (8/10)
  → Crisis Agent 즉시 활성화
  → 안전 계획 안내 + 1393 연결 제안
  → 전문가 대시보드 즉시 알림
  → 감사 로그 기록
```

### 시나리오 3: Expert-in-the-loop
```
고위험 세션에서 AI 응답 → 전문가 대기열 진입
  → 정신건강의학과 전문의가 응답 수정
  → 수정된 응답이 사용자에게 전달
  → 피드백이 향후 에이전트 프롬프트에 반영
```

### 시나리오 4: 다회기 컨텍스트 유지
```
2주 후 재방문 사용자
  → 장기 메모리에서 이전 핵심 이슈 검색
  → "지난번에 말씀하셨던 업무 스트레스는 어떻게 되셨나요?"
  → 치료 이력 기반 맞춤 접근
```
